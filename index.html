Buds for SUDS

about

Suds for Buds is an beer companion app that uses jQuery and API data to pair meal options with complimentary craft beers. Submit a food-related term in the form below to receive a list of appropriate results.

This app was developed by Edward Bacal and Vigyan Kayastha, using data from punkapi.com, and designed by Edward Bacal.
